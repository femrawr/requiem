## Audio (audio)
Plays an audio on the device. The audio cannot be stopped or paused when it is playing.

## Brightness (brightness)
Sets the device's brightness.

## Blue Screen of Death (bsod)
Triggers the blue screen of death, crashing the device.

## Critical (critical)
Makes Requiem a critical process/windows process. An attempt at killing Requiem will trigger the Blue Screen of Death and crash the device.

## Download (download)
Downloads a file or multiple files to the device. It will download to temp if a path is not set. The names of the file or files will always be the unix epoch.

## File (file)
Does various actions on a file or directory.

## Input (input)
Blocks or unblocks inputs to the device.

## Jumpscare (jumpscare)
Displays a picture on the screen for a set ammount of time. The picture will be fullscreened.

## Macro (macro)
Lets you create a macro to run multiple commands at once and much more.

## Message Box (msgbox)
Displays a Message Box on the screen.

## Persist (persist)
Repersists Requiem if its persistence has been disabled.

## Ping (ping)
Simple test command to see if Requiem is online.

## Process (process)
Blocks or unblocks a process from being ran.

## Rotate (rotate)
Rotates the screen's display.

## Run (run)
Runs a provided process. If the first argument is cmd or powershell, it will automatically insert /c or -c before any other arguments.

## Screenshot (ss)
Takes a screenshot of the device's screen.

## Settings (settings)
Configure the runtime settings.

## Site (site)
Blocks or unblocks a website from being accessed.

## Tree (tree)
Displays a file tree showing all files and sub-directories in a directory.

## Text to Speech (tts)
Plays a text-to-speech message on the device. It cannot be stopped or paused when it is playing.

## User account control (uac)
Checks if Requiem is elevated, bypasses UAC, and disabled UAC related settings.

## Update (update)
Removes Requiem and replaces it with a provided file.

## Upload (upload)
Uploads a file or directory to the server.

## Volume (volume)
Sets the device volume.

## Wallpaper (wallpaper)
Sets the device wallpaper.

## Webcam (webcam)
Takes a picture from the device's webcam.

## Wipe (wipe)
Removes Requiem from the device. It will automatically unpersist and will turn Requiem into a normal process if it is critical.

---

## Help (help)
Lists all the available commands with their description and the command prefix.

## List (list)
Every active session of Requiem will reply to this with the channel linked. It will work in any channel the bot has access to.

## Command prefix
Runs the last command it received with its message context.
